파이썬 업무자동화

개발환경
os : windows 10 Pro
lang : python 3.9, python 3.6
IDE : pycharm
package : PyQt5, openpyxl, pyinstaller

요구사항

  1. 배차 운행 기록을 엑셀에 입력합니다.
  2. 사용자는 50대로 엑셀 다루는 능력이 없습니다.
  3. 그래픽 UI를 이용하여 데이터를 입력하고 싶습니다.
  4. 잘못된 정보를 입력할 경우 입력이 최소 되거나 경고로 알립니다.
